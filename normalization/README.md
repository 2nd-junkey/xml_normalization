xmlの正規化が面倒なのでプログラムを組む

# 前提条件

- 個人を特定されない事
- 手作業クソだるい
- 現場にあるものでどうにかする(c#)



 # 仕様

- xmlを取り込む
- コマンドラインで片付ける
  - `~.exe source.xml dest.xml`
    - 新規追加
  - `~.exe -i source.xml`
    - 既存ファイル上書き
- 子ノードになる度インデント(1tab)
- 属性は改行してインデント(2tab)
  - 閉じ括弧は開き括弧と同じインデント
- 子ノードも属性も持たない場合だけ一行ノード



# 設計

- xmlファイル(引数)読み込み
- 出力先確認
- ルートノード(深度0)から処理
  - 名前
  - 属性
  - 子ノード
